
Важно

Бэкенд должен быть доступен по адресу:

http://localhost:1452/

Требования

- Node.js (рекомендуется 18+)
- npm или pnpm

Запуск локально (Windows `cmd.exe`)

1) Установите зависимости (в корне проекта):

```cmd
npm install
```

или если вы используете pnpm:

```cmd
pnpm install
```

2) Запустить в режиме разработки (Vite):

```cmd
npm run dev
```
